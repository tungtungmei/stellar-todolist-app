# Stellar TodoList DApp

## Description

This project is a simple Todo List application built using a smart contract on the Stellar blockchain with Soroban.

It allows users to store and retrieve tasks directly on the blockchain, removing the need for a centralized database.

---

## Features

* Add new tasks
* View all stored tasks
* Data stored on blockchain (decentralized)

---

## How It Works

* Tasks are stored using smart contract storage
* Data is saved in a vector (array-like structure)
* Users interact with the contract through provided functions

---

## Smart Contract Functions

### add_task

Adds a new task to the blockchain

### get_tasks

Returns all stored tasks

---

## How to Use

1. Deploy the smart contract
2. Call `add_task` to add a task
3. Call `get_tasks` to view tasks

## Conclusion

This project demonstrates a simple use of blockchain technology for storing and managing data in a decentralized way.
